# Entity Framework Core - Proje Uygulaması

Bu proje, C# ve .NET kullanarak veri tabanı işlemlerini yönetmek amacıyla geliştirilmiş bir uygulamadır. Veri erişim katmanında **Entity Framework Core** kullanılmıştır.

## 🚀 Projenin Amacı
Bu uygulama ile CRUD (Create, Read, Update, Delete) işlemlerini EF Core üzerinden gerçekleştirerek veri tabanı yönetimini modernize etmeyi hedefledim.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** C#
* **Framework:** .NET / Entity Framework Core
* **Veritabanı:** SQL Server
* **Araçlar:** Visual Studio, Git/GitHub

## 📋 Özellikler
* Veri tabanı tablolarının Model (Entity) olarak tanımlanması.
* LINQ sorguları ile veri listeleme ve filtreleme.
* Context yapısı üzerinden veri tabanı bağlantı yönetimi.
